# 🧠 Sign Language Generation AI

## 📌 Project Title
**Sign Language Generation for Visual Aid of Special Person**

---

## 📖 Description
A machine learning-based system that converts **spoken or typed input into sign language visuals (images/GIFs)** to assist communication for hearing-impaired individuals.  

The project leverages **Natural Language Processing (NLP)** and **Deep Learning models** including **TF-IDF + Logistic Regression** and **RNN-LSTM**, integrated into a **Flask-based web application** for an interactive user experience.

---

## 🚀 Features
- 🔤 Text → Sign Language Conversion  
- 🎤 Speech Input Support  
- 🧠 TF-IDF + Logistic Regression Model  
- 🔁 RNN-LSTM Model for sequential learning  
- 🌐 Interactive UI using Flask  

---

## 🛠️ Tech Stack
- **Programming Language:** Python  
- **Machine Learning:** TF-IDF, Logistic Regression, RNN, LSTM  
- **Framework:** Flask  
- **Frontend:** HTML, CSS  
- **Libraries:** NumPy, Pandas, Scikit-learn  

---

## 🧠 Models Used

### 1️⃣ TF-IDF + Logistic Regression
- Converts textual data into numerical feature vectors  
- Performs classification to map words to corresponding sign labels  

### 2️⃣ RNN-LSTM Model
- Handles sequential and contextual data  
- Improves prediction accuracy for phrases and sentences  

---

## ⚙️ How It Works
1. User provides input (text or speech)  
2. Input is preprocessed (cleaning & tokenization)  
3. Model predicts the corresponding sign language label  
4. System outputs relevant image or GIF
