# sign-language-generation-ai

Sign Language Generation for Visual Aid of Special Person

Developed an intelligent sign language generation system that converts spoken or typed input into visual sign representations using NLP and deep learning models (TF-IDF, Logistic Regression, RNN-LSTM), integrated into a user-friendly Flask web application.

📌 Features
🔤 Text → Sign Language Conversion
🎤 Speech Input Support
🧠 TF-IDF + Logistic Regression Model
🔁 RNN-LSTM Model for sequence learning
🌐 Interactive UI using Flask

🛠️ Tech Stack
Python
Machine Learning (TF-IDF, Logistic Regression, RNN, LSTM)
Flask (Web App)
HTML/CSS
NumPy, Pandas, Scikit-learn

🧠 Models Used
1. TF-IDF + Logistic Regression
Converts text into numerical vectors
Classifies words into corresponding sign labels

2. RNN-LSTM Model
Handles sequential data
Improves prediction for phrases/sentences

⚙️ How It Works
User inputs text or speech
Text is preprocessed
Model predicts corresponding sign
System displays image/GIF
